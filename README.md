# Praia X Smash

Jogo estilo Whack-A-Mole feito em Unity para Android.

## Instalação

Baixe o whack.apk [clicando aqui](https://github.com/GabrielMendesMelo/Praia-X-Smash/raw/main/whack.apk) para jogar em um celular Android.
